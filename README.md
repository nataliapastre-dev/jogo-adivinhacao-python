# 🎮 Jogo de Adivinhação em Python

Projeto desenvolvido em **Python** com interface gráfica utilizando **Tkinter**.

O objetivo do jogo é adivinhar o número secreto gerado aleatoriamente pelo sistema.

---

## 🚀 Funcionalidades

- 🎯 Sistema de dificuldade (Fácil, Médio e Difícil)
- 🔢 Contador de tentativas
- 📜 Histórico de palpites
- 🏆 Sistema de recorde
- 🎮 Interface gráfica com imagem
- ⌨️ Suporte para tecla **Enter** para jogar

---

## 🛠 Tecnologias utilizadas

- Python
- Tkinter
- Pillow

---
